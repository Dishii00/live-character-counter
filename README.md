# Experiment 2 - Live Character Counter

## Course Outcome Mapped
CO1 – Explain the fundamental concepts of front-end and back-end development, including HTML, CSS, JavaScript, React.js, Node.js, and databases.

## Aim
To implement a live character counter that updates as users type in a text area.

## Objectives
- Create a responsive text input component
- Implement real-time character counting
- Add visual feedback for character limits
- Style with modern CSS techniques

## About the Program
This experiment demonstrates reactive UI updates using React state hooks and CSS transitions for real-time feedback. The character count updates instantly as the user types in the text area. It also provides visual feedback when the character limit is near or reached.

## Features
- Real-time character counting
- Responsive text area
- Warning and limit feedback
- Smooth CSS transitions
- Modern and clean user interface

## Hardware/Software Requirements

### Software
- VS Code
- React 18+
- Chrome DevTools

### Hardware
- i3+ CPU
- 4GB RAM

## Learning Outcomes of Experiment
- Learned how to use React state for dynamic updates
- Understood real-time rendering in React
- Implemented character counting logic
- Applied CSS transitions for better UI
- Built a responsive front-end component

## Steps to Run the Program
1. Open the project folder in VS Code
2. Run `npm install`
3. Run `npm run dev`
4. Open the localhost link in browser
